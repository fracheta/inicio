📄 O que é o arquivo INDEX.HTML?

O index.html é o arquivo principal de qualquer site — é como a "porta de entrada" de um projeto web. Quando você acessa um site, o navegador procura automaticamente por esse arquivo para exibir o conteúdo.

📁 meu-portfolio/
  ├── 📄 index.html ← Página principal (meus projetos)
  ├── 📄 welcome.html ← Esta tela de boas-vindas
  ├── 🎨 style.css
  └── 📜 script.js

No meu portfólio, o index.html contém todos os meus projetos organizados — os scripts JavaScript, as apresentações, os apps mobile e muito mais. É lá que você vai encontrar tudo o que eu já desenvolvi durante meus estudos.
